# Paradise Nursery Shopping Application

This is the final project for the React and Redux course: Paradise Nursery Shopping App.

## Features
- Paradise Nursery landing page with company name and "Get Started" button.
- AboutUs page displaying company details.
- Product listing page with at least 6 unique plants per category.
- Shopping cart implemented with Redux (CartSlice).
- Navbar with links to Home, Plants, and Cart, with cart icon count.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/panchamipammi123-maker/e-plantShopping.git
   cd e-plantShopping
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the app:
   ```bash
   npm start
   ```
